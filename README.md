# Python Basic Programs 
This repository contains simple and easy-to-understand Python programs
created for learning and practicing programming basics.
All programs are written in a beginner-friendly way and are suitable for
1st and 2nd year students.
---
## Folder Structure
### basics/
Basic beginner programs covering conditions, operations, and simple logic.
- Sample program
- Addition of two numbers
- Largest and second largest of three numbers
- Even or odd check
- Positive, negative or zero check
- Swapping two numbers (with and without temporary variable)
- Multiplication table
- Unit conversion (kilo to pounds)
---
### numbers/
Programs related to numerical logic and calculations.
- Factorial of a number
- Fibonacci series
- Prime number check
- Prime numbers within a range
- Reverse of a number
- Palindrome number
- Sum of digits
- Count number of digits
- GCD of two numbers
- LCM of two numbers
- Sum of natural numbers
---
### strings/
Programs related to characters and string logic.
- Check vowel, consonant, digit or special character
- Display corresponding character for a given integer
- Display corresponding integer for a given character
---
### sequences/
Programs based on number sequences and series.
- 1, 2, 4, 7, 11, 16, ...
- 1, 2, 4, 8, 16, ...
- 1, 3, 9, 21, 41, ...
---
## How to Run the Programs
1. Install Python on your system
2. Download or clone this repository
3. Navigate to the required folder
4. Run a program using:
   ```bash
   python filename.py
